Sistemas para Internet

Este profissional elabora projetos e administra redes de computadores, instala e configura equipamentos e softwares, configura servidores web e constrói bancos de dados para a internet. Monta sites também para redes internas das empresas (intranet) e trabalha na implantação de sistemas de comércio eletrônico. A frente de trabalho mais dinâmica está no desenvolvimento de aplicativos para smartphones e tablets. Matemática, empreendedorismo, criação de páginas para web e linguagens de programação são algumas das disciplinas que integram a grade curricular. Algumas graduações se voltam para a parte gráfica da construção de sites (webdesign), enquanto outras reforçam aspectos técnicos da construção e montagem de páginas.
